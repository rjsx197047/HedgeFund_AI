# Screenshots

Drop PNG captures of the live app here. The root [`README.md`](../../README.md) references these paths:

| File | What it should show |
|---|---|
| `analyze.png` | The Analyze page mid-debate. Ideally: NVDA selected, a provider picked (e.g. OpenAI · gpt-4o-mini), a few agents already streamed in (analysts done, researchers debating), the cost line visible. |
| `settings.png` | The Settings page with at least one provider connected (green pill) and the OpenAI OAuth row showing "Connected as ... · plus plan" (or similar). Hide the actual email if you'd rather. |
| `history.png` | The History page list view showing 3-5 past sessions of different tickers. Bonus: detail pane open with one transcript visible. |
| `watchlist.png` | The Watchlist page with 4-6 tickers added. |

## Capture tips

- **Window size:** ~1280 × 800 looks crisp without forcing horizontal scrolling on narrow GitHub layouts.
- **Theme:** Dark theme is the v1 brand signature — keep it dark. Don't switch to light mode for screenshots.
- **Privacy:** Blur or replace your real OAuth email + ChatGPT plan tier if you'd prefer not to publish them. Replace with `you@example.com` and `plus plan` placeholders if needed.
- **Format:** PNG. JPEG compression artefacts on dark UI screenshots look bad on retina displays.
- **Filename:** Lowercase, hyphenated, matches the table above exactly so the README references stay valid.
